# Proces uruchamiania aplikacji

## Windows

### Wymagania systemowe

- python w wersji 3 dostępny z linii poleceń pod etykietą `python`
- manager pakietów `pip`

### Proces uruchamiania

Należy uruchomić skrypt `run.bat`

## Linux

### Wymagania systemowe

- python w wersji 3 dostępny z linii poleceń pod etykietą `python`

### Proces uruchamiania aplikacji

Należy uruchomić skrypt `run.sh`.
